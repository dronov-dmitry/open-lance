# Как посмотреть список пользователей в удаленной базе MongoDB Atlas

У вас есть несколько очень удобных способов посмотреть список всех пользователей, так как вы используете удаленную базу данных MongoDB Atlas. 

Поскольку мы ранее создали endpoint, вы можете сделать это прямо через ваш сайт, либо подключиться к базе напрямую несколькими путями:

## Способ 1: Прямо через ваш сайт (самый простой)
1. Запустите ваш фронтенд (используя `npm start` или `scripts\start_dev.ps1`).
2. Зайдите на сайт (обычно `http://localhost:8080`, в зависимости от настроек) и **авторизуйтесь** под любым созданным аккаунтом.
3. В верхнем меню нажмите на новую вкладку **«Фрилансеры»**. 
4. Ваш локальный фронтенд сделает легитимный API-запрос к `https://open-lance-backend.<ваш-субдомен>.workers.dev/users` с секретным токеном, и выведет список всех профилей на экран.

---

## Способ 2: Через официальный сайт базы данных (MongoDB Atlas)
Эта база данных "лежит" в облаке. Вы можете посмотреть вообще все "сырые" данные: пароли (хеши), email'ы, даты регистрации.
1. Зайдите на сайт [cloud.mongodb.com](https://cloud.mongodb.com/) и залогиньтесь.
2. Откроется панель вашего кластера. Нажмите на кнопку **"Browse Collections"** (Посмотреть коллекции).
3. Слева в списке баз данных выберите базу `open-lance`.
4. Внутри нее кликните на коллекцию **`users`**.
5. На экране появится полный список всех документов (пользователей) в вашей удаленной базе в формате JSON.

---

## Способ 3: Через программу MongoDB Compass (Для профессиональной работы)
Если вы хотите управлять удаленной базой прямо с компьютера, как файлами в папке, не открывая браузер — скачайте официальный клиент:
1. Скачайте и установите **MongoDB Compass** (с официального сайта MongoDB).
2. Откройте программу. На главном экране будет поле `URI`.
3. Скопируйте туда вашу строку из файла `.env` (например: `mongodb+srv://<пользователь>:<пароль>@<кластер>.mongodb.net/...`).
4. Нажмите **Connect**. 
5. Слева выберите базу `open-lance`, затем `users`. Вы увидите удобную таблицу со всеми пользователями. Там же их можно будет редактировать или удалять одним кликом.

---

## Способ 4: Через Postman / Insomnia / curl (API Запросы)
Если хочется дернуть API напрямую из специализированной консоли:
1. Сделайте **POST-запрос** на `https://open-lance-backend.<ваш-субдомен>.workers.dev/auth/login` с JSON-телом: 
   ```json
   {
     "email": "ваш-email", 
     "password": "ваш-пароль"
   }
   ```
2. В ответ бекенд вернет вам строку: `{"success": true, "token": "eyJhb..."}`.
3. Затем сделайте **GET-запрос** на `https://open-lance-backend.<ваш-субдомен>.workers.dev/users`.
4. К этому запросу в программе прикрепите Header:
   `Authorization: Bearer тут_тот_самый_длинный_токен`
5. Вы получите JSON-ответ со списком всех пользователей без паролей (так, как они отображаются на сайте).
